🚨 TROUBLESHOOTING:

Problem: "No candidates gathered in trickle-ice"
Solution:
  1. Check internet connection
  2. Try different STUN server (use Google first)
  3. Check firewall not blocking UDP port 3478
  4. Try from mobile hotspot (different ISP)

Problem: "High latency (>200ms)"
Solution:
  1. Use faster server from test results
  2. Move closer to WiFi router
  3. Close background apps
  4. Switch to 5G network

Problem: "Connection drops frequently"
Solution:
  1. Enable TURN relay mode
  2. Use fallback tier servers
  3. Check which servers work in your region
  4. Monitor connection quality metrics

Problem: "Connection never establishes"
Solution:
  1. Extreme firewall (corporate)
  2. ISP blocks P2P connections
  3. Try TURN relay (costs bandwidth but works)
  4. Check if port ranges blocked (30000-65000)
